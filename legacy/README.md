## ✨ 주요 기능

### 🎯 핵심 의사소통
- **286개 픽토그램 카드**: 동사 중심 + 핵심어휘(core vocabulary) 포함
- **280개 커스텀 SVG 픽토그램**: 한국 문화에 최적화된 디자인
- **12개 카테고리**: 핵심, 동작, 감정, 사람, 장소, 음식, 사물, 시간, 질문, 상황, 학교, 긴급
- **수정된 Fitzgerald Key 색상 체계**: 카테고리별 색상 코딩 (WCAG AA 준수)
- **문장 조합**: 카드 다중 선택 → 자연스러운 한국어 문장 생성
- **긴급 표현 바**: 아파요, 도와주세요, 화장실 가요 등 빠른 접근

### 📱 메인 메뉴 시스템
- **4기능 메인 메뉴**: 스플래시 → 메인 메뉴(말하기/상황/기록/설정) → 각 기능
- **직관적 네비게이션**: 색상 구분된 4개 정사각형 타일 + 하단 탭 바
- **로고 터치 → 메인 메뉴 복귀**: 어디서든 쉽게 돌아가기

### 🏠 상황판 (9개 상황)
- 집, 병원, 식당, 학교, 어린이집, 차, 공원, 마트, 목욕
- 상황별 맞춤 카드 세트 (9~16개씩)
- 컬러 그라디언트 카드 + 이모지 아이콘

### 🔊 음성 출력 (TTS)
- Web Speech API 기반 한국어 음성
- 카드 선택 시 자동 발화
- 전체 문장 읽기 + 반복 재생 (3회, 5초 간격)
- 음성 선택 (기기 내장 한국어 음성)
- Chrome 15초 제한 자동 우회

### 👀 크게 보기 (청자 모달)
- 선택된 카드를 전체 화면으로 표시
- 픽토그램 + 텍스트 대형 표시
- 긴급 상황 시 붉은 배경 강조

### ♿ 스캐닝 접근성 (4단계)
- **메뉴 단계**: 말하기/상황 선택
- **카테고리 단계**: 카테고리 탭 또는 상황 카드 순환
- **카드 단계**: 개별 카드 순환 + ⬅돌아가기
- **모달 단계**: 크게보기 화면의 ✕닫기 버튼 스캐닝
- 자동 모드 / 단계별 모드
- 속도 조절 (0.5~5초)
- 하이라이트 색상 커스터마이즈
- 모바일 터치 오버레이 (이중 발화 방지)
- 2바퀴 순환 시 상위 단계로 자동 복귀

### 📊 사용 기록 & 통계
- 의사소통 빈도 그래프 (상위 8개 카드)
- 총 사용 횟수, 단어 수, 최다 사용 카드 표시
- 기록 + 통계 일괄 삭제
- 히스토리 타임라인 (시간별 발화 기록)

### 🔍 검색
- 헤더 검색 팝업 (카드 이름으로 검색)
- 검색 결과 클릭 → 해당 카테고리 이동 + 카드 시안 발광 효과

### ⚙️ 설정
- 고대비 모드 / 다크 모드
- 글꼴 크기 조절 (작게/보통/크게/아주크게)
- 진동 피드백 ON/OFF
- 스캐닝 설정 (모드, 속도, 색상)

### 🎨 접근성
- WCAG AA 색상 대비 (4.5:1 이상)
- 모든 버튼 aria-label 적용
- 키보드/스위치 접근 가능
- 움직임 감소 설정 대응 (prefers-reduced-motion)
- 반응형 레이아웃 (320px~1024px)

## 🚀 실행 방법

### 로컬 실행
```bash
# 1. 저장소 클론
git clone https://github.com/your-username/allinone-aac.git
cd allinone-aac

# 2. 로컬 서버 실행 (ES Modules 사용으로 서버 필수)
# Python
python3 -m http.server 8080

# Node.js
npx serve .

# VS Code
# Live Server 확장 설치 후 index.html 우클릭 → Open with Live Server

# 3. 브라우저에서 접속
# http://localhost:8080
```

### GitHub Pages 배포
```bash
# Settings → Pages → Source: main branch → Save
# https://your-username.github.io/allinone-aac/
```

> ⚠️ **주의**: `file://` 프로토콜에서는 ES Modules가 작동하지 않습니다.
> 반드시 HTTP 서버를 통해 실행하세요.

| 카테고리 | 색상 | 주요 카드 |
|---------|------|---------|
| 🔴 핵심 | 빨강 | 네/아니요, 원해요, 먹어요, 있어요/없어요, 잘, 알아요 |
| 🟢 동작 | 초록 | 먹어요, 마셔요, 가요, 씻어요, 놀아요 |
| 🔵 감정 | 파랑 | 좋아요, 싫어요, 행복해요, 슬퍼요, 무서워요 |
| 🟡 사람 | 노랑 | 엄마, 아빠, 선생님, 친구, 의사 |
| 🟠 장소 | 주황 | 집, 학교, 병원, 마트, 공원 |
| 🟤 음식 | 갈색 | 밥, 빵, 물, 과자, 과일 |
| ⚪ 사물 | 회색 | 옷, 신발, 책, 연필, TV |
| 🟣 시간 | 보라 | 지금, 나중에, 오늘, 내일, 곧 |
| 🔵 질문 | 청록 | 뭐예요?, 어디?, 언제?, 왜? |
| ⬜ 상황 | 연보라 | 병원, 학교, 식당 |
| 🏫 학교 | 남색 | 숙제, 책, 공부해요, 쉬는 시간 |
| 🔴 긴급 | 빨강(강조) | 아파요, 도와주세요, 화장실 가요 |

## ♿ 스캐닝 사용법

### 4단계 스캐닝 흐름
```
스캐닝 ON
    ↓
[메뉴 단계] 말하기 ↔ 상황
    ↓ 선택
[카테고리 단계] 카테고리 탭 순환 (또는 상황 카드 순환)
    ↓ 선택
[카드 단계] ⬅돌아가기 → 카드1 → 카드2 → ...
    ↓ 카드 선택
[모달 단계] 크게보기 화면 → ✕닫기 버튼 스캔
    ↓ 닫기
[카드 단계 복귀] 같은 카테고리 계속 선택 가능
```

### 자동 복귀
- 카드 2바퀴 순환 → 카테고리 단계로 자동 복귀
- 카테고리 2바퀴 순환 → 메뉴 단계로 자동 복귀

### 입력 방식
| 입력 | 동작 |
|------|------|
| 화면 터치 | 모바일 터치 오버레이 "선택" 버튼 |
| 스페이스/엔터 | PC 키보드 선택 |
| 블루투스 스위치 | 키보드로 인식되어 동일하게 동작 |
