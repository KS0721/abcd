# 올인원 AAC — 의사소통 약자를 위한 보완·대체 의사소통 앱

> **"말로 표현하기 어려운 사람들이, 카드 터치만으로 자신의 말을 할 수 있도록."**

## 왜 만들었는가

뇌병변, 자폐 스펙트럼, 지적장애, 후천적 언어장애 등으로 **구어 의사소통이 어려운 사람들**은 일상에서 가장 기본적인 의사 표현조차 막히는 경우가 많습니다.

기존 AAC 앱은 대부분 **영어권 중심**이거나, **고가의 전용 기기**를 필요로 하거나, 한국어 문법 지원이 부족합니다. 특히 한국어는 어순과 조사 체계가 영어와 달라, 단순히 카드를 나열하는 것만으로는 자연스러운 문장이 되지 않습니다.

**올인원 AAC**는 이 문제를 해결하기 위해 만들었습니다.
- 스마트폰이나 태블릿만 있으면 **무료로** 사용 가능
- **한국어 문법에 맞는 자동 정렬**로 자연스러운 문장 생성
- 언어치료 현장에서 검증된 **Fitzgerald Key 색상 체계** 적용
- 인터넷 없이도 작동하는 **PWA + 오프라인 TTS**

---

## 무엇을 만들었는가

### 핵심: 카드를 골라 문장을 만들고, 소리로 전달한다

사용자가 카드(그림 + 글자)를 터치하면 문장이 조립되고, 음성으로 출력됩니다.

**사용 흐름:**
1. 카테고리 선택 (사람, 동작, 감정, 음식 등)
2. 원하는 카드 터치 → 상단 문장 바에 추가
3. 한국어 문법에 맞게 **자동 정렬** (주어 → 목적어 → 동사)
4. 재생 버튼 또는 자동으로 **음성 출력**

## 주요 기능

### 말하기 — 500개 어휘 카드

14개 카테고리에 걸친 **500개 기본 어휘 카드**로 다양한 문장을 만들 수 있습니다.

| 카테고리 | 색상 | 예시 |
|---------|------|------|
| 표현 | 핑크 | 네, 아니요, 안녕, 고마워요 |
| 동작 | 초록 | 먹어요, 가요, 씻어요, 놀아요 |
| 감정 | 파랑 | 좋아요, 싫어요, 행복해요, 슬퍼요 |
| 사람 | 보라 | 엄마, 아빠, 선생님, 친구 |
| 음식 | 주황 | 밥, 빵, 물, 과자, 우유 |
| 장소 | 청록 | 집, 학교, 병원, 공원 |
| 사물 | 주황 | 옷, 신발, 가방, 핸드폰 |
| 시간 | 갈색 | 지금, 나중에, 오늘, 내일 |
| 동물 | 연두 | 강아지, 고양이, 토끼 |
| 자연 | 초록 | 비, 해, 꽃, 나무 |
| 신체 | 살색 | 머리, 배, 손, 발 |
| 색깔 | 무지개 | 빨강, 파랑, 노랑, 초록 |
| 묘사 | 하늘 | 크다, 작다, 뜨거워요, 차가워요 |
| 의료 | 빨강 | 아파요, 약, 주사, 열나요 |

각 카테고리는 **Fitzgerald Key 색상 체계**를 따릅니다. 이는 언어치료 분야에서 품사별 색상 코딩으로 문법 구조를 시각적으로 구분하는 국제 표준 방식입니다.

### 긴급 카드 — 위급 상황 즉시 전달

**6개 긴급 카드**는 항상 화면 상단에 고정되어, 위급 상황에서 한 번의 터치로 도움을 요청할 수 있습니다.
- 도와주세요 / 아파요 / 그만해요 / 화장실 / SOS / 싫어요
- 빨간 배경 + 경고 아이콘으로 시각적 강조

### 상황판 — 20가지 실생활 장면

병원, 식당, 학교 등 **20가지 실생활 상황**에 맞는 카드 세트를 미리 구성했습니다. 상황을 선택하면 그 장소에서 가장 많이 쓰는 12개 카드가 바로 나타납니다.

| 상황 | 핵심 표현 예시 |
|------|--------------|
| 병원 | 여기 아파요, 무서워요, 약 주세요 |
| 식당 | 배고파요, 물 주세요, 맛있어요, 더 주세요 |
| 학교 | 선생님, 모르겠어요, 도와주세요, 쉬고 싶어요 |
| 마트 | 이거 사고 싶어요, 과자, 무거워요 |
| 차 안 | 다 왔어요?, 속 안 좋아요, 창문 열어주세요 |
| 미용실 | 짧게 잘라주세요, 감아주세요, 다 됐어요 |
| 치과 | 이 아파요, 무서워요, 다 끝났어요? |
| 수영장 | 도와주세요, 물, 샤워해요 |

그 외 집, 어린이집, 공원, 목욕탕, 카페, 운동, 여행, 영화관, 도서관, 약국, 직장 상황판 포함.

### 크게 보기 — 대화 상대에게 보여주기

카드를 길게 누르면 **전체 화면으로 확대**되어, 소리를 들을 수 없는 환경에서도 화면을 보여주며 의사소통할 수 있습니다.
- "이 화면을 보여주세요" 안내 문구 표시
- 긴급 카드는 붉은 배경으로 주의 환기
- TTS 자동 재생 + 반복 재생 옵션

### 카드 관리 — 나만의 카드 만들기

기본 카드 외에 **사용자가 직접 카드를 만들 수 있습니다.**
- 텍스트 입력 + 카테고리 선택
- **ARASAAC 픽토그램 자동 검색** — 단어를 입력하면 관련 그림을 자동으로 찾아줍니다
- **카메라 촬영** — 실제 사물을 촬영해 사진 카드로 등록
- 카드 순서 드래그 앤 드롭으로 변경

### 발화 기록 — 자주 쓰는 말 빠르게

- **기록 화면**: 최근 100개 발화 내역을 저장, 터치하면 다시 재생
- **빠른 말**: "안녕하세요", "감사합니다", "도와주세요" 등 자주 쓰는 문장을 즐겨찾기

### 음성 듣기 (STT) — 상대방 말을 글자로

상대방의 말을 **음성 인식으로 텍스트 변환**하여 화면에 표시합니다. 청각장애가 있는 사용자도 대화 내용을 확인할 수 있습니다.

## 접근성 — 누구나 사용할 수 있도록

이 앱의 사용자는 손을 자유롭게 쓰지 못하거나, 시각에 제한이 있거나, 인지 능력이 다양한 분들입니다. 그래서 접근성은 부가 기능이 아니라 **설계의 출발점**입니다.

### 자동 스캐닝
손으로 정확한 위치를 터치하기 어려운 사용자를 위해, 카드가 **3초 간격으로 자동 하이라이트**됩니다.
- 1단계: 카테고리 순회
- 2단계: 카드 순회
- 3단계: 크게 보기 + 음성 출력
- **터치 한 번 = 선택**, 꾹 누르기 = 끄기, 두 번 터치 = 다음 화면

### 체류 시간 (Dwell Time)
떨림이 있는 사용자의 오작동을 방지하기 위해, 카드를 **일정 시간 누르고 있어야 선택**되도록 설정할 수 있습니다. (0ms / 500ms / 1000ms)

### 시각 접근성
- **고대비 모드** 기본 활성화
- **다크 모드** 지원
- **글자 크기 3단계** 조절 (중 / 대 / 특대)
- **카드 격자 크기** 5단계 조절 (2×2 ~ 6×6)
- 색상에만 의존하지 않는 정보 전달 (아이콘 + 텍스트 병행)

### 촉각 피드백
카드 선택 시 **진동 피드백**으로 선택이 이루어졌음을 알려줍니다.

## 설계 철학

### 왜 그림 카드인가
AAC 분야의 핵심 원칙은 **"그림은 언어를 대신할 수 있다"**는 것입니다. 글자를 읽지 못하는 사용자도 그림을 보고 의미를 이해할 수 있습니다. 이 앱은 국제 표준 픽토그램인 **ARASAAC**(아라곤 AAC 센터)의 그림 기호를 사용하며, 김치·떡볶이·비빔밥 등 **한국 고유 음식 31종은 직접 제작한 SVG 아이콘**으로 대응했습니다.

### 왜 Fitzgerald Key인가
1920년대 청각장애 교육에서 시작된 Fitzgerald Key는, 품사별 색상 코딩으로 **문법 구조를 시각적으로 이해**하게 돕는 방법입니다. 언어치료사들이 실제 현장에서 사용하는 검증된 체계이며, 이 앱은 이를 디지털로 구현했습니다.

### 왜 오프라인이 중요한가
의사소통 보조 도구는 **인터넷이 없는 환경에서도 반드시 작동해야 합니다.** 병원 진료실, 이동 중인 차 안, 시설 내부 등 네트워크가 불안정한 곳에서도 음성 출력이 가능하도록 **로컬 TTS 엔진**(ONNX 기반)을 탑재했으며, PWA로 오프라인 설치를 지원합니다.

## 아키텍처

**도메인 주도 설계(DDD)**를 적용하여 6개 핵심 도메인으로 분리했습니다.

| 도메인 | 역할 |
|--------|------|
| **Card** | 어휘 카드 데이터, 카테고리, 사용자 카드 관리 |
| **Sentence** | 한국어 문법 규칙에 따른 문장 조합 |
| **Situation** | 20가지 상황판 데이터 및 카드 세트 |
| **Speech** | TTS 엔진 관리, 음성 설정 (속도·높낮이·볼륨) |
| **Scanning** | 자동 스캐닝 로직 (3단계 순회) |
| **User-Data** | 발화 기록, 빠른 말, 데이터 백업/복원 |
